Changelog

All notable changes to git-switchboard. Follows Keep a Changelog and Semantic Versioning.

v0.3.0 2025-12-15 Latest
New 4
Improved 2
Fixed 3
Release v0.3.0
New
  • PR dashboard with live CI status — launch with git switchboard pr to see all open PRs across your repos with pass/fail/pending indicators
  • Retry CI directly from the PR dashboard by pressing R on any selected PR — triggers a new run without leaving the terminal
  • Clone a PR's branch locally from the PR dashboard with c — sets up tracking and optionally opens in your editor immediately
  • Review status now displayed in the branch picker — shows approved, changes requested, or pending review state alongside CI status
Improved
  • Fuzzy search performance significantly improved on repos with 500+ branches — now uses an indexed trie structure for sub-millisecond filtering
  • Refresh behavior after checkout no longer briefly flickers the branch list — transitions are now frame-synchronized
Fixed
  • Crash on launch when no GitHub token is configured — now degrades gracefully with a clear inline notice rather than a panic
  • Stale branch list displayed after checkout in some edge cases — now always re-fetches from git after any branch change
  • Incorrect sort order on branches with identical commit timestamps — now uses a stable secondary sort by branch name

New 4

Release v0.2.0
New
  • GitHub integration — authenticate with a personal access token to show linked PR numbers and states directly in the branch picker
  • CI checks display in branch picker — pass/fail/pending CI status shown inline for branches with open pull requests
  • --author <user> flag — filter branch list to a specific author at launch without entering the TUI filter
  • --remote / -r flag — include remote branches in the picker listing, shown with a remote prefix
v0.1.0 2025-06-01 Initial Release
New 6
Platforms 3
Release v0.1.0
Status First
New
  • Initial release — keyboard-driven TUI for listing and checking out git branches without typing branch names
  • Interactive branch picker with author, last-updated, and branch name columns sorted by recency
  • Fuzzy search with / key — instantly narrows branch list as you type, no confirmation needed
  • Vim keybindings (j/k to navigate, gg/G to jump to top/bottom) alongside arrow key support
  • Cross-platform binary distribution — single executable for macOS (Intel + Apple Silicon), Linux (x86_64 + ARM), and Windows
  • Homebrew tap and Chocolatey package for one-line installation on macOS/Linux and Windows respectively
