GitHub integration requires a personal access token with repo scope. Set GITHUB_TOKEN or configure via git switchboard config.

Tip

Press / to enter fuzzy search mode. Matches branch names, authors, and PR titles simultaneously.

Warning

Checking out a branch with uncommitted changes will prompt you to stash or discard before switching.

Danger

Force checkout with --force will discard all uncommitted changes without confirmation.
