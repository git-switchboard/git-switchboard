Branch Navigator · PR Dashboard · GitHub Integration

Subsystems

SUBSYSTEM 01
Interactive Branch Picker

Keyboard-driven TUI for listing, filtering, and checking out branches. Fuzzy search, vim keybindings, and real-time author filtering. Never type a branch name again.

SUBSYSTEM 02
PR Dashboard

Full pull request overview across all your repos. Track CI status, review state, and open PRs without leaving the terminal. Press ↵ to open in browser.

SUBSYSTEM 03
GitHub Integration

Authenticate once with a GitHub token and get PR status, CI checks, and review states overlaid directly on the branch list. Remote and local branches unified.

SUBSYSTEM 04
Cross-Platform

Ships as a single binary for macOS, Linux, and Windows. Install via Homebrew, Chocolatey, or a shell script. Zero runtime dependencies — just run it.
