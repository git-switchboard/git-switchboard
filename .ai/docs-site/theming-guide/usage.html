Usage Reference

Complete CLI reference for git switchboard. All commands, flags, keybindings, and configuration options.

Commands

git-switchboard adds a single git switchboard command with two subcommands.

git switchboard Launch the branch picker TUI
-r, --remote Include remote branches in the listing
--author <user> Filter branch list to a specific author on launch
-h, --help Print help and exit
-V, --version Print version and exit
git switchboard pr Launch the PR dashboard TUI
--repo <name> Filter to a specific repository by name
--all Show pull requests from all accessible repositories
-h, --help Print help and exit
Info

Run git switchboard --help or git switchboard pr --help at any time to see current flag options in your installed version.

Branch Picker

The branch picker is the default TUI launched by git switchboard. It displays all local (and optionally remote) branches with author, last-updated, and linked PR data in a scrollable table.

The status bar at the top shows current filter state. The current branch is highlighted in teal. Branches with associated PRs show the PR number and state inline.

PR Dashboard

Launch with git switchboard pr. Shows open pull requests across all repos you have access to, with CI status and last-updated timestamps at a glance.

Columns shown: PR number, title, repository, CI status (✓/✗/○), and last updated. Press ↵ to open the selected PR in your default browser.

Tip

Use --repo my-repo-name to scope the PR dashboard to a single repository — useful when working across many active projects.

Configuration

git-switchboard reads configuration from ~/.config/git-switchboard/config.toml. Create it manually or run git switchboard config to open it in your editor.

~/.config/git-switchboard/config.toml 
# GitHub integration
[github]
token = "ghp_..."

# UI defaults
[ui]
show_remote = false
default_author = ""

Config Fields

Key Type Default Description
github.token string "" GitHub personal access token with repo scope
ui.show_remote bool false Include remote branches in the picker by default
ui.default_author string "" Pre-filter branches by this author on launch. Leave empty for all authors.
Warning

Never commit your github.token to source control. Alternatively, set the GITHUB_TOKEN environment variable instead of writing it to the config file.
